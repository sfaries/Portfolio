Description

This project focused on using jQuery AJAX API to make calls to stackOverflow's stackExchange API. The idea is to help new developers build a reputation on StackOverflow by quickly identifing unanswered questions based on searchable tags. First, I forked some starter files and focused on writing the code necessary to return objects in JSON to the console. Then I focused on transforming the object data into html.
